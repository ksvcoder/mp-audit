# Скрипт для разбора Google таблицы

Этот скрипт разбирает столбец E на отдельные строки и повторяет для каждого значения данные из столбцов A-D.

## Результат

Исходные данные (пример строки 2):
```
A: 1
B: Приоритет 1 (Фундамент и выживание)
C: Информационная безопасность
D: ISO/IEC 27001/17/18
E: Вопрос 1; Вопрос 2; Вопрос 3
```

Результат (3 строки):
```
A: 1  | B: Приоритет... | C: Информационная безопасность | D: ISO/IEC... | E: Вопрос 1
A: 1  | B: Приоритет... | C: Информационная безопасность | D: ISO/IEC... | E: Вопрос 2
A: 1  | B: Приоритет... | C: Информационная безопасность | D: ISO/IEC... | E: Вопрос 3
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите Google Sheets API ключ:
   - Перейдите на https://console.cloud.google.com
   - Создайте новый проект
   - Включите Google Sheets API
   - Создайте сервисный аккаунт (Service Account)
   - Скачайте JSON ключ в формате "Private Key"
   - Сохраните как `credentials.json` в этой папке

3. Дайте доступ к таблице:
   - Откройте Google таблицу
   - Нажмите "Поделиться"
   - Скопируйте email сервисного аккаунта из файла `credentials.json` (поле "client_email")
   - Дайте доступ с правами редактирования

## Использование

```bash
python process_sheets.py
```

Скрипт автоматически:
1. Прочитает данные из исходного листа
2. Разберёт столбец E (поддерживает разделители: новая строка, точка с запятой, запятая)
3. Создаст новый лист "результат"
4. Загрузит туда развёрнутые данные

## Структура

- **Исходный лист**: первый лист в таблице (имя определяется в переменной `SOURCE_SHEET`)
- **Лист результата**: "результат" (создаётся автоматически если не существует)
- **Заголовки**: берутся из первой строки исходного листа

## Примечания

- Скрипт автоматически определяет, какой разделитель используется в столбце E
- Пустые значения игнорируются
- Остальные столбцы (кроме A-E) также копируются в результат
