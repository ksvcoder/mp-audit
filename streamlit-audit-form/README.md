# IT Audit Checklist - Streamlit форма

Веб-форма для заполнения IT аудита с автоматическим сохранением ответов в Google Sheet.

## Функции

✅ Показывает вопросы из таба "результат"  
✅ Просит имя пользователя  
✅ Выбор ответа (Да/Нет/Частично)  
✅ Условное отображение поля для факта (при Да/Частично)  
✅ Поле для комментариев  
✅ Автоматическое сохранение ответов в новый таб "ответы"  
✅ Навигация (назад/далее/пропустить)  

## Локальное использование

### 1. Установите зависимости:
```bash
cd streamlit-audit-form
pip install -r requirements.txt
```

### 2. Положите credentials.json в папку

Получить Google API ключ: см. инструкцию в `pandas-tab1/README.md`

### 3. Запустите форму:
```bash
streamlit run app.py
```

Откроется в браузере на `http://localhost:8501`

## Развертывание на Streamlit Cloud (БЕЗ СЕРВЕРА!)

### Шаг 1: Загрузить код на GitHub

```bash
git init
git add .
git commit -m "Add IT audit form"
git remote add origin https://github.com/YOUR_USERNAME/mp-audit.git
git push -u origin main
```

### Шаг 2: Развернуть на Streamlit Cloud

1. Перейти на https://share.streamlit.io
2. Нажать **"New app"**
3. Выбрать:
   - GitHub repo: `your-username/mp-audit`
   - Branch: `main`
   - File path: `streamlit-audit-form/app.py`
4. Нажать **"Deploy"**

### Шаг 3: Добавить Google API ключ в Streamlit Secrets

1. На странице приложения нажать **⚙️ Settings** → **Secrets**
2. Скопировать содержимое `credentials.json`
3. В поле "Secrets" вставить:

```toml
[gcp_service_account]
type = "service_account"
project_id = "..."
private_key_id = "..."
private_key = "..."
client_email = "..."
client_id = "..."
auth_uri = "..."
token_uri = "..."
auth_provider_x509_cert_url = "..."
client_x509_cert_url = "..."
```

4. Нажать **Save** → приложение перезагрузится

### Шаг 4: Поделиться ссылкой

Ссылка будет типа: `https://mp-audit-form.streamlit.app`

Отправь эту ссылку сотрудникам - они смогут заполнять форму!

## Решение проблем

**❌ "Ошибка при загрузке данных"**
- Проверьте, что Google Sheet имеет таб "результат"
- Проверьте, что сервис-аккаунту дан доступ к таблице

**❌ "credentials.json не найден"**
- Для локального запуска: положите файл в папку проекта
- Для Streamlit Cloud: добавьте в Secrets (см. Шаг 3 выше)

## На сколько усложняет добавление имени?

**На 0%!** 😄 Это буквально 5 строк кода:
- 1 строка: `st.text_input()` для ввода имени
- 1 строка: сохранить в `session_state`
- 3 строки: передать имя в функцию сохранения

Всё остальное работает автоматически.
